# 3D Portfolio

An interactive 3D portfolio website built with React, Three.js, and Framer Motion, featuring immersive 3D models and smooth animations.

## Overview

A modern developer portfolio with interactive 3D elements rendered using React Three Fiber. Features animated sections, 3D model displays, and a contact form powered by EmailJS.

## Features

- **3D Models** — Interactive Three.js scenes with GLTF models
- **Smooth Animations** — Framer Motion page transitions
- **Contact Form** — EmailJS integration for direct messaging
- **Responsive Design** — Tailwind CSS responsive layout
- **React Router** — Multi-page navigation

## Tech Stack

- **React** 18 — UI framework
- **Three.js** / **React Three Fiber** — 3D rendering
- **@react-three/drei** — Three.js helpers
- **Framer Motion** — Animations
- **Tailwind CSS** — Styling
- **EmailJS** — Contact form
- **Vite** — Build tool

## Getting Started

```bash
git clone https://github.com/okekefrancis112/3d-portfolio.git
cd 3d-portfolio
npm install
npm run dev
```

### Build

```bash
npm run build
npm start
```
